- simple_zone
  ゾーンタイプとして、店だけを考慮
  GPUで性能向上を図ったが、、、

- complex_zone
  ゾーンタイプとして、店だけでなく、レストラン、工場などを追加。
  キューをglobalメモリに置くことで、性能が少しアップした。
  キューを全スレッドで共有するようにしてみたところ、性能は変わらず。しかも、バグあり。キューへの競合のせい？なので、却下した。
  それでも、なんだかんだで、200x200で、CPUより速い。
  CPU版: 18秒
  CUDA版: 5.7秒

- shared_zone
  shared memoryを使用してみよう！
  
  CPU版: 18秒
  GPU版: 2秒 (近似解を計算する場合)
  GPU版: ～30秒(各ブロックの周辺部についても、距離マップを更新することで、厳密解に近づける)

  ⇒とりあえず、近似解で進めよう。性能的には、これ以上の向上は難しいと思う。。。

- my_mcmc
  shared_memoryを元に、MCMCを実装。
